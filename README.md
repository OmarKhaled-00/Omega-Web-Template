## 📌 Frontend Web Template

A modern and responsive frontend web template built using HTML and CSS.  
This project focuses on clean design, structured layout, and user-friendly interface.

### 🚀 Features
- Responsive design for all screen sizes
- Clean and minimal UI
- Well-structured HTML & CSS
- Easy to customize

### ⚠️ Note
This is a static template (UI only) and does not include any backend or dynamic functionality.
